SIL765: Network & System Security


Home   –   Course Description   –   Administrivia   –   Reading List   –   Schedule   –   Lab   –   Other Links   

4 credits (3-0-2)

Course Objectives

  • Learn principles of building secure networks and systems
  • Learn security challenges in existing and emerging networks
  • Analyze existing network protocols from an adversarial standpoint
  • Build new secure networking protocols

Syllabus

  • Foundations of Information Security
  • Cryptographic Hash Function
  • Symmetric Key Cryptography
  • Asymmetric Key Cryptography
  • Network Security
  • Key Distribution
  • Access Control
  • Electronic Mail (Email) Security
  • Domain Name Server (DNS) Security
  • Transport-Level Security (HTTPS, SSH)
  • Internet Protocol Security (IPSec)
  • Virtual Private Network (VPN)
  • Firewall
  • Wireless Network (WLAN, Bluetooth, 5G) Security
  • Physical Layer Security
  • System Security
  • Malware
  • Program Analysis
  • Penetration Testing
  • Embedded System and Hardware Security
  • Intrusion Detection System (IDS)

  • Security of Evolving Technologies
  • Adversarial Machine Learning
  • Cloud Security
  • Biometric Security
  • Blockchain and Peer-to-Peer Payments (such as Bitcoin)
  • Software-Defined Networking Security
  • Privacy and Anonymous Communication Networks (such as Tor)

Prerequisites

  • Basic Networks: COL334: Computer Networks
  • Basic Cryptography : COL759: Cryptography
  • Basic Programming : Python

Textbook Information

Reference Book: Network Security Essentials Applications and Standards by William Stallings

Assignements

There will be five assignments and one project.You may collaborate when solving the assignments, however when writing up the solutions you must do so on your own.You must submit all assigned work electronically. Submission instructions will be provided.
